# ドキュメント索引

この `docs/` ディレクトリには、Raspberry Pi から M5Stack Core2 を操作するためのワークフロー・要件・テスト・トラブルシュートを体系的にまとめています。以下の表に目を通し、必要な章へジャンプしてください。

| ファイル名 | 内容の概要 |
|-----------|------------|
| `development-environment.md` | `arduino-cli` やライブラリのインストール手順、接続済みデバイス情報など、開発実行環境の整備方法を説明しています。
| `dev-guide.md` | Pi ↔ Core2 開発のチェックリスト、実行済みコマンド、次に取り組むべきステップをまとめたガイドです。手順どおりに作業を進めたいときに参照します。
| `interface-requirements.md` | JSON シリアルインタフェースの仕様（受け取るフィールド、使用する表情、CLI/Web側とのやり取り）を記述しています。StackChan顔のみを使うことにも触れています。
| `tests.md` | TDD に従ったテストマトリクス。実施済みのコマンドと結果、未完了項目を記録し、次のテストにつなげるための場です。
| `journal.md` | 日々の作業ログ。重要な変更、ユーザ報告、テスト状況をタイムスタンプ入りで記録しています。
| `troubleshooting.md` | 発生した問題とその解決策を番号付きで追記しているログです。コンパイルエラー／スクリプトのタイムアウトなどの参考に。

新しいドキュメントを追加する場合は、この README に行を追加して一覧を最新に保ってください。
